# Jobify: AI-Based Job & Internship Recommendation System
 
**Jobify** is a production-ready, highly scalable modern web application that leverages sophisticated NLP and Machine Learning techniques (TF-IDF, Cosine Similarity, Named Entity Recognition) to intelligently match candidates with tech employers based on skill gap analyses and resume parsed data.

## Features

- **Candidate Panel**: Upload PDF/DOC mock resume files for automated skill extraction using **spaCy**. Features a smart AI-powered Job Match dashboard with one-click Job ID copying for analysis.
- **AI-Powered Chatbot**: Integrated **Gemini AI** (using Google GenAI SDK) to provide instant career guidance and interview coaching for candidates.
- **Skill Gap Analysis**: Deep analysis of missing skills between candidate profile and job requirements, featuring dynamic **YouTube tutorial suggestions** via API.
- **Employer Panel**: Clean interface to post open positions with specific **Job Types** (Full Time, Internship, etc.) and track candidate matches dynamically scoring applicant suitability.
- **Admin Panel**: Manage the integrity of the platform (users, jobs) via analytics tools powered by **Recharts**.
- **Modern & Premium UI**: Re-engineered with a sleek, consistent **Pill-style Design System**. Transitions and micro-animations powered by **Framer Motion** for a state-of-the-art SaaS experience.

## Tech Stack
- Frontend: `React (Vite)`, `Tailwind V4`, `Framer Motion`, `Recharts`
- Backend: `Python`, `FastAPI` (Uvicorn), Pydantic
- Database: `MongoDB` (PyMongo)
- NLP/AI: `spaCy`, `scikit-learn`, `pdfplumber`

## 🛠 Installation Instructions

Ensure you have **Python 3.10+**, **Node.js 20+**, and a locally running **MongoDB Compass** daemon (defaulting to `localhost:27017`).

### 1. Backend Setup

Open a terminal window and navigate to the project directory:

```bash
cd backend

# Create a virtual environment (Optional but Recommended)
python -m venv venv
venv\Scripts\activate
# Windows: venv\\Scripts\\activate
# Mac/Linux: source venv/bin/activate

# Install all Python dependencies
pip install -r requirements.txt

# Download required spaCy model for NLP Parsing
python -m spacy download en_core_web_sm

# start the server
python -m uvicorn main:app --reload
```
# Spin up the development server

The server will initialize natively at `http://localhost:8000`. Full autogenerated API documentation is available at `http://localhost:8000/docs`.

### 2. Frontend Setup

Open a new, separate terminal window:

```bash
cd frontend

# Install package dependencies (Vite, React, Tailwind UI ecosystem)
npm install

# Start the interactive UI
npm run dev
```

Visit `http://localhost:5173` in your browser.

## Recommended Workflow for Testing
1. Click **Sign Up** using the **employer** role. Create a test user: `google`, `google@test.com`, with any password.
2. In the Employer Dashboard, click `Post a Job`. E.g., title `React Engineer`, requirements `React, Javascript, CSS`.
3. Open a private tab (or log out) and hit **Sign Up** using the **candidate** role. 
4. Upload a mock `.pdf` file with tech keywords inside via the **Upload Resume** tab. See the AI parse out skills.
5. Head to the **Recommendations** tab and watch the engine provide matches against previously established jobs.

## Folder Structure Highlights

- `backend/ai_modules` : Contains the deep algorithm pipelines for recommendation engine, TF-IDF calculation, and string-gap analytics.
- `backend/routes` : Organized separation of logical routing (candidate, admin, auth).
- `frontend/src/pages` : Nested routing UI structure segregating user types efficiently for JWT session validations.
